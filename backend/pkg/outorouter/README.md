# OutoRouter


## 概要

![ロゴ](outorouter.png)

### 名前の由来

**開発界隈の無法者「Vide Coding」を用いた8割自動体験 x マグロの大トロって気がついたら溶けてるよね**

- LLMコーディングエージェントを用いた 開発プロセスをほぼ自動化するGo言語新世代ルーター
- 気がついたらコードが出来上がっている、まるで大トロを食べるかのような開発者体験




### 機能

- ✅ 各種エンドポイント(handler)のリクエストモデル, レスポンスモデルのクライアント用の型を自動生成
  -  Go, TypeScript, Dartに対応
  - `router/router.go`に定義されたルーティング情報から各種handlerの情報を取得
- ✅ アクセスログ, エラーログを自動で出力
- ✅ 各種ミドルウェアの自動適用
  - 認証, CORS, ロギング, リクエストID付
- ✅ 独自のAPIドキュメントを生成 (Reactで実装予定)
  - PostmanのようなUIでAPIを試せる
- ✅ HTTP2 / gRPC Streaming / WebSocket / HTTP3 / QUIC 対応
- ✅ E2Eテストコードの自動生成 
  - 各種エンドポイントに対するE2Eテストコードを自動生成
  - E2Eテストの雛形を生成することで、テストコードの作成工数を削減
